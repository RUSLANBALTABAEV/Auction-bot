# Руководство по развертыванию Auction Bot

## Требования к системе

### Минимальные требования
- **ОС**: Windows 10/11 (64-bit)
- **Память**: 4 GB RAM
- **Процессор**: 2-ядерный CPU 1.8 GHz+
- **Место на диске**: 1 GB свободного места
- **Интернет**: Стабильное широкополосное соединение

### Рекомендуемые требования  
- **ОС**: Windows 11 (64-bit)
- **Память**: 8 GB RAM
- **Процессор**: 4-ядерный CPU 2.5 GHz+
- **Место на диске**: 2 GB свободного места
- **Интернет**: Высокоскоростное соединение с низкой задержкой

### Необходимое ПО
- [Python 3.9+](https://www.python.org/downloads/)
- [NCALayer](https://ncl.pki.gov.kz/) (для ЭЦП)
- Активный сертификат ЭЦП
- Telegram (для уведомлений, опционально)

## Установка

### Способ 1: Автоматическая установка (Windows)

1. **Скачайте последнюю версию**
   - Получите архив `auction-bot-vX.X.X.zip` из раздела releases
   - Распакуйте архив в папку `C:\auction-bot\` (или другую удобную)

2. **Запустите установщик**
   - Дважды щелкните на `install.bat`
   - Дождитесь завершения установки зависимостей
   - При необходимости подтвердите установку браузера

3. **Настройте конфигурацию**
   - Запустите `config_generator.py` или
   - Отредактируйте `config.yaml` вручную
